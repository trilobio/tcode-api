Pipette Tip Layout
==================

The :class:`~tcode_api.schemas.pipette_tip_layout.PipetteTipLayout` schema describes the
arrangement of pipette tips within a tip box, used when issuing pipette tip commands.

Related T-code Commands:
- :class:`~tcode_api.schemas.commands.ADD_PIPETTE_TIP_GROUP`
